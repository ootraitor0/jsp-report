# 웹페이지 구현 계획하기
<개요>
 1. 쇼핑몰 웹 사이트 개발를 개발합니다
 2. 판매 할 목록을 추가,업데이트,삭제할 수 있는 기능을 구현합니다
    - 동시에 구매자가 상품을 검색해서 구매 할 수 있도록 설계 합니다
 3. 데이터베이스로 Oracle 11g을 이용합니다
 4. 디자인과 접근 방법을 반영하고 문서를 작성합니다
 5. 웹페이지 구현시 사용하는 JSP,JAVA Servlet, JAVABeann 기술을 사용 합니다
 
 <구체적인 준비물>
  1. Java와 JSP 소스 코드
  2. Oracle 데이터베이스에서 사용되는 SQL 스크립트
  3. 환경 설정 파일 및 문서
  4. 잘 작성된 프로젝트 설명문
    - Java 클래스 파일들의 관계를 보여주는 UML클래스 다이어그램을 포함한 전체적인 디자인 설명
    - 어떻게 구현할 것인지 에 대한 묘사
    - 각 과정들의 테스트에 대한 스크린샷 결과물과 테스트 데이터 포함하는 계획서
    - 결과 분석
