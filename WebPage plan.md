# 웹페이지 구현 계획하기
<개요>
 1. 쇼핑몰 웹 사이트 개발를 개발합니다
 2. 판매 할 목록을 추가,업데이트,삭제할 수 있는 기능을 구현합니다
    - 구매자가 상품을 검색해서 구매 할 수 있도록 설계 합니다
 3. 데이터베이스로 Oracle 11g을 이용합니다
 4. 디자인과 접근 방법을 반영하고 문서를 작성합니다
 5. 웹페이지 구현시 사용하는 JSP,JAVA Servlet, JAVABeann 기술을 사용 합니다
 
 <준비물>
  1. Java와 JSP 소스 코드
  2. Oracle 데이터베이스에서 사용되는 SQL 스크립트
  3. 환경 설정 파일 및 문서
  4. 잘 작성된 프로젝트 설명문  
    - Java 클래스 파일들의 관계를 보여주는 UML클래스 다이어그램을 포함한 전체적인 디자인 설명
    - 어떻게 구현할 것인지 에 대한 묘사
    - 각 과정들의 테스트에 대한 결과물과 계획서
    - 결과 분석

 <문서 형식>
  1. 표지 및 참고를 포함을 시키되 쪽수 범위에 포함시키지 않습니다.
  2. 글자 크기 : 12 / 폰트 : 맑은 고딕
  3. 페이지의 여백 : 1inch(2.54cm)
  4. 단락의 여백(줄 간격) : 2.0
  5. 그 외 참고 문서 및 테이블 등등은 적절하게 맞춰서 작성합니다

 <프로그래밍 형식>

  1. 블럭 형식의 머리에 주석 작성합니다
  2. 각 코드에 대한 주석을 작성합니다
  3. 적절한 변수와 함수 이름을 작성합니다
  4. 적절한 들여쓰기를 이용합니다

  <참고>
  1. 프로젝트 설계는 다른 것을 참고할 수는 있지만 표절은 하지 않습니다
  2. 프로젝트 완성하게되면 사용된 파일을 압축하고, 문서를 정리해 보관합니다

 <요구사항>
  1. 쇼핑몰에는 2명의 사용자가 있습니다. (관리자,고객)
  2. 관리자는 데이터베이스에 생성된 관리자 계성을 사용합니다
    - 고객은 데이터베이스에 계정이 없으므로 회원가입 기능을 이용하여 사용합니다
